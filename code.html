function getValues() {
    let trustInput = document.getElementById('trustInput');
    let worthyInput = document.getElementById('worthyInput');
    let startNumber = parseInt(trustInput.value);
    let endNumber = parseInt(worthyInput.value);
    if (Number.isInteger(startNumber) && Number.isInteger(endNumber)) {
        let generatedNumbers = generateTrustworthy(startNumber, endNumber);
        displayNumbers(generatedNumbers);
    }
    else {
        
        Swal.fire({
            backdrop: false,
            title: 'oops',
            text: 'please enter valid start and end number',
            icon: 'error',
        });
    }
}



function generateTrustworthy(start, end) {
    
    let numbers = [];


    for (let i = start; i <= end; i = i + 1) {
        if (i % 3 == 0 && i % 5 == 0) {
            numbers.push("TrustWorthy");
        }

        else if (i % 3 == 0) {
            numbers.push("Trust")
        }
        else if (i % 5 == 0) {
            numbers.push("Worthy")
        }

        else
            numbers.push(i);
    }   
    return numbers;
}



function displayNumbers(numbers) {

    let tableHtml = '';

    for (let index = 0; index < numbers.length; index = index + 1) {
        let currentNumber = numbers[index];
        let className = '';

        tableHtml = tableHtml + `<tr><td class="${className}"> ${currentNumber}</td></tr>`;
    }

    let tableElement = document.getElementById('trustworthyTable');
    tableElement.innerHTML = tableHtml;

   


}

How it works

Similar to the loop challenge TrustWorthy has three functions.

getValues()

First, we take a look at the getValues function. we start by extracting the input id from our html by using the function let trustInput = document.getElementId('trustInput') after we extract both input id's from our Html, we get the value out of the input by creating new values using javascript. The values that we will be using in js is startNumber and endNumber our startNumber will be used for our trustInput and the endNumber will be used as our worthyInput. i.e. let startNumber = parseInT(trustInput.value); as stated previously in the loop challenge, parseInt allows you to turn a string of text into numbers and not just any number, parseInt allows you to turn a string of text into integers. we will now use an if statement to validate that the input is indeed in fact integers by using the function if (number. isInteger(startNumber) && Number.isInteger(endNumber)) is true then we go to the next step, which is generate numbers. If either the startNumber/endNumber comes back as false then the sweet alert will show in the browser, alerting the user to enter valid numbers.

generateTrustworthy()

Now that we know that the inputs entered by the user are both integers, we will run a for loop that after looping will take those numbers and return them back to the getValue function. Our for loop function states that if our start number is less than our end number, the loop will continue by adding one until the start number is no longer less than the end number. Furthermore, our if statements that are apart of our loop helps us identify Trustworthy (numbers that are divisible by 3 and 5 with no remainders), Trust (numbers that are divisible by three with no remainders), Worthy (numbers that are divisible by five with no remainders), and regular integers that are neither divisible by three or five. Note: to replace the numbers that are divisible, with words, we must incorporate else if. The array of numbers and phrases will be returned back to the getValues function where they will be displayed by what our last function is.

displayNumbers

Our last function we will be using is displayNumbersthis function isnt a built-in js function but a custom function that we're using to display an array of numbers and phrases on our Trustworthy webpage. First we will run another for loop. The let index = 0; intitializes a variable index to 0 which is the starting point, next index < numbers.length: is the condition that keeps the loop running. it checks to make sure that the current index is less than the total number of elements in the array. lastly is index = index + 1 this will increase the index by one until the loop ends because the index is no longer less than the total number of elements inside the array. currentNumber and className are both properties that helps us create the table for html the function tableHtml = tableHtml + `<tr><td class="${className}"> ${currentNumber}</td></tr>`; is now used as the tableElement id that we have extracted from our html which is trustworthyTable. This will allow javascript to be interactive with our html without having to write any of the functions for js on our html page.
